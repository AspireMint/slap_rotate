# slap_rotate  


Easiest way to rotate nodes? :-) 
Slap node with glove! 
  
Glove recipe (like screwdriver, but uses wool instead of steel ingot):  
{ group:wool }  
{ group:stick }  
  
Usage:  
- left-click (dig) rotates node horizontally  
- right-click (place) rotates node vertically
